## üí° Strategic Rationale

### Why Free Core Platform?

1. **Lower Barrier to Entry**
   - Farmers can try platform at no cost
   - Build trust and engagement
   - Learn system before committing

2. **Data Collection**
   - Platform gains production data
   - Improve AI/ML models
   - Better insights for all users

3. **Government Partnership**
   - Support agricultural digitization
   - Help small farmers modernize
   - Social impact

4. **Network Effects**
   - More farmers = better procurement matching
   - Richer data ecosystem
   - Stronger platform

### Why Paid Marketplace?

1. **Premium Value**
   - Marketplace generates direct revenue for farmers
   - Customer acquisition channel
   - Worth paying for

2. **Quality Control**
   - Subscription filters serious commercial farmers
   - Reduces spam/inactive listings
   - Better buyer experience

3. **Sustainable Revenue**
   - Predictable monthly income
   - Scales with farmer success
   - Funds platform maintenance and growth

4. **Flexible Monetization**
   - Farmers only pay if they need marketplace
   - No forced subscription
   - Can downgrade anytime

## üîê Technical Implementation Notes

### Subscription Check Middleware

```python
def check_marketplace_access(user):
    """
    Check if user has active marketplace subscription.
    """
    farm = user.farm
    
    # Check if farm has active subscription
    if not farm.subscription:
        return False, "No marketplace subscription"
    
    subscription = farm.subscription
    
    # Check subscription status
    if subscription.status in ['active', 'trial']:
        return True, "Marketplace access granted"
    
    elif subscription.status == 'past_due':
        days_left = subscription.days_until_suspension
        return True, f"Grace period: {days_left} days remaining"
    
    else:  # suspended, cancelled
        return False, "Marketplace subscription inactive"
```

### Feature Access Control

```python
class MarketplaceFeature:
    """Decorator to require marketplace subscription"""
    
    def __init__(self, feature_name):
        self.feature_name = feature_name
    
    def __call__(self, func):
        def wrapper(request, *args, **kwargs):
            has_access, message = check_marketplace_access(request.user)
            
            if not has_access:
                return Response({
                    'error': 'Marketplace subscription required',
                    'message': message,
                    'upgrade_url': '/subscriptions/plans/'
                }, status=403)
            
            return func(request, *args, **kwargs)
        
        return wrapper
```
